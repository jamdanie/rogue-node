# Rogue Node: Data Center Containment

A browser-based cybersecurity stealth and investigation game built with **TypeScript, Phaser, and Vite**.

**Rogue Node** puts players inside a compromised data center where they must investigate suspicious activity, avoid patrol detection, analyze infrastructure, inspect evidence, and contain a rogue system before sensitive information is leaked.

## 🕹️ Gameplay Preview

You play as an investigator navigating a rogue data center while attempting to stop a malicious system before a simulated data leak completes.

Players must:

- Verify badge access
- Investigate employee identity
- Avoid roaming patrols
- Analyze server logs
- Recover laptop evidence
- Inspect network patch panels
- Review SOC terminal alerts
- Analyze camera risks
- Contain the **Rogue Node**

All while managing:

🚨 Threat level  
⏱️ Data leak timer  
👮 Patrol detection  
🧠 Cybersecurity decision-making

## 🚀 Run Locally

Clone the repository:

```bash
git clone https://github.com/jamdanie/rogue-node.git
```

Open the project:

```bash
cd rogue-node
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the local URL shown in the terminal (typically):

```bash
http://localhost:5173
```
