# Rogue Node: Data Center Containment

Rogue Node is a browser-based cybersecurity training game built with TypeScript and Phaser. The player investigates a fictional rogue data center by avoiding patrols, reviewing evidence, analyzing SOC alerts, inspecting infrastructure, and containing an unauthorized data leak.

## Live Demo

Coming soon via GitHub Pages.

## Gameplay Features

- Cybersecurity-themed 2D stealth gameplay
- Mission timer and data leak countdown
- Roaming guard with proximity detection
- Threat meter and camera detection zone
- Locked security door with badge/identity verification
- Server rack log investigation
- Laptop evidence recovery
- Randomized SOC terminal scenarios
- CIA Triad and incident response questions
- Rogue Node final containment objective
- Procedural pixel-art assets created in code
- Sound effects, lighting, and cyber atmosphere

## Cybersecurity Concepts

This project demonstrates concepts from information assurance, networking, and physical security:

- Confidentiality, Integrity, Availability
- Social engineering awareness
- Badge access and tailgating risks
- Physical security patrols
- Camera blind spots
- Network cabling and patch panels
- Suspicious outbound traffic
- Log review and incident response
- Evidence collection before containment
